# To-Do List Web App

A simple and responsive task management application built using HTML, CSS, and JavaScript.

## Objective
To build a task management application that allows users to organize daily tasks efficiently.

## Features
- Add New Tasks
- Delete Tasks
- Mark Tasks as Completed
- Persistent Data Storage using localStorage
- Responsive User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- localStorage

## Functionality
Users can:
- Add tasks to the list
- Mark tasks as completed
- Remove unwanted tasks
- View saved tasks even after refreshing the page
